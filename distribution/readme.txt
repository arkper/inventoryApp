Instructions to install and configure the application:

1. Download and unpack the application archive inventoryApp-master.zip into a local directory inventoryApp
2. Install Microsoft SQL Server Compact Edition by running executable inventoryApp-master\distribution\dependencies\SSCERuntime_x86-ENU.exe
3. Install Microsoft Access/Excel driver by running executable inventoryApp-master\distribution\dependencies\AccessDatabaseEngine.exe
4. The application is ready to execute under inventoryApp-master\distribution\InventoryWiz.exe
